Web Poll Application

A focus on backend development using Django

This was the first application I made where a user arrives at the poll questions page, selects a question, selects an answer from set choices and then is taken to a results view where the total votes per choice are displayed. Users can then click to Vote Again. I also have an admin page where questions can be viewed, edited and new questions added.
